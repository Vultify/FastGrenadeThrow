# Fast Grenade Throw

Grenade keybinds that actually throw the grenade — press, throw, back to your gun. No weapon swap dance.

Overhand and underhand each get their own bind. Grenades come from anywhere on you: rig, pockets, armband, backpack, modded equipment slots included. If Use Items Anywhere is installed, it hooks into that instead of fighting it.

Keybinds and a debug toggle live in F12.

**Install:** release zip into your SPT root folder — lands as `BepInEx/plugins/FastGrenadeThrow/FastGrenadeThrow.dll`. SPT 4.0.x / BepInEx 5.x.
