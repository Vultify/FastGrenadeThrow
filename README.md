# Fast Grenade Throw

Instant grenade throws for SPT with dedicated overhand and underhand keybinds — no weapon swap dance.

## Features

- Quick throw on its own keybinds, separate binds for overhand and underhand
- Grabs grenades from rig, pockets, armband, and backpack — modded equipment slots included
- Auto-detects Use Items Anywhere and hooks into it conflict-free
- F12 config: keybinds, debug logging toggle

## Install

Extract the release zip into your SPT root folder — it lands as `BepInEx/plugins/FastGrenadeThrow/FastGrenadeThrow.dll`.

## Compatibility

- SPT 4.0.x
- BepInEx 5.x
